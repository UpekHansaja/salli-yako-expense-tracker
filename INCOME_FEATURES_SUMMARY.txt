================================================================================
                              DATABASE SCHEMA
================================================================================

monthly_income table:
  - id (INTEGER PRIMARY KEY)
  - user_id (INTEGER, FOREIGN KEY)
  - amount (DECIMAL 10,2)
  - renewal_date (INTEGER 1-31)
  - last_renewal_date (DATE)
  - next_renewal_date (DATE)
  - description (TEXT)
  - active (INTEGER 0/1)
  - created_at (DATETIME)
  - updated_at (DATETIME)

other_income table:
  - id (INTEGER PRIMARY KEY)
  - user_id (INTEGER, FOREIGN KEY)
  - amount (DECIMAL 10,2)
  - source (TEXT)
  - description (TEXT)
  - income_date (DATE)
  - category (TEXT)
  - created_at (DATETIME)
